Create a new feature wish with guided brainstorming.

**Input**: Optionally provide a description of what you want (e.g., `/opsx-wish I want dark mode`).

**Steps**

Invoke the `openspec-wish` skill. Pass any provided input as the initial description.

**Output**
After the skill completes, summarize:
- Wish name and location
- Affinity level
- Next step: run `/opsx-engage` to start working on it, or leave for the daemon
